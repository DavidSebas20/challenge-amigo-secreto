# 🎉 Amigo Secreto 🎁

## Descripción
Aplicación web para realizar sorteos de "Amigo Secreto" de forma rápida y sencilla. Permite agregar participantes, ver la lista y realizar el sorteo aleatorio.

## Características
- ✅ Agregar nombres de amigos de forma dinámica
- ✅ Validación de campos vacíos
- ✅ Interfaz intuitiva con diseño moderno
- ✅ Resultado del sorteo con animación de confeti
- ✅ Listado en tiempo real de participantes

## Instalación
1. Clona este repositorio:
```bash
git clone https://github.com/tu-usuario/amigo-secreto.git](https://github.com/DavidSebas20/challenge-amigo-secreto.git
cd challenge-amigo-secreto
```
2. Esta es la estructura de archivos:
```bash
📁 challenge-amigo-secreto/
│── 📄 index.html   # Página principal con la interfaz del usuario
│── 📄 style.css    # Estilos de la aplicación
│── 📄 app.js       # Lógica del programa
│── 📁 assets/      # Imágenes e íconos utilizados en la página
```
## Uso
1. Abre el archivo index.html en tu navegador
2. Ingresa nombres en el campo de texto y haz clic en "Añadir"
3. Verás la lista de amigos actualizándose automáticamente
4. Haz clic en "Sortear amigo" para obtener el resultado aleatorio

## Posibles problemas y soluciones
- El botón de sorteo no funciona: Asegúrate de haber agregado al menos un amigo a la lista antes de sortear.
- No se muestra la lista de amigos: Revisa si el archivo app.js está correctamente enlazado en index.html.
- Los estilos no se aplican: Verifica que style.css esté correctamente referenciado en el <head> del HTML.
